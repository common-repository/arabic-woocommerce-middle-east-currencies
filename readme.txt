Installing **Arabic Woocommerce Middle east currencies** will add Arabic full translation to WooCommerce plugin (always update - up-to-date) ,Additional Middle east currencies like Saudi Riyal ,Bahrain, Kuwait, Dinar, Oman, Qatar, Rial ,Egypt Pound and Providing Arab support to solve any Issues relate Woocommerce in Arabic

**Currently compatible with version `2.6+` of WooCommerce**

* Arabic full translation to WooCommerce plugin (always update - up-to-date)

== Installation ==

**From your WordPress Dashboard**

To install this plugin directly from your WordPress dashboard:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

or

* Log in to your WordPress dashboard.
* Navigate to the "Plugins" menu and click "Add New".
* In the search field, type “Arabic Woocommerce Middle east currencies” and click "Search Plugins".
* Once you’ve found the plugin, you can install it by simply clicking “Install Now”.


**Manually**

The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

**Updating**

Each time **WooCommerce** gets updated, a new version of **Arabic Woocommerce Middle east currencies** will get updated as well, so it keeps the language files up-to-date.
